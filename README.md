# iNotebook - Secured Notes on Cloud
## **Description** : 
* Secure
* Stored on cloud
* Notes categorised on basis of tags
* Notes can be edited or deleted
* Global Search feature gives results on basis of title, description and tags
* Completely free
___
## **Tech Stack** : 
* HTML
* CSS
* JavaScript
* MongoDb
* Express Js
* Node Js
* React Js
* Bootstrap
* Git / GitHub
